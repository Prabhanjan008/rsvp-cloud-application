# RSVP Cloud Application

## Overview
A cloud-based RSVP web application deployed using AWS services. Users submit RSVP details via a web form, which are stored in an RDS MySQL database through a Node.js backend hosted on EC2.

## Tech Stack
- AWS EC2
- AWS RDS (MySQL)
- Node.js
- Express.js
- HTML

## Features
- Form submission
- Cloud database storage
- Backend deployment on AWS

## Key Learnings
- Cloud deployment
- EC2 and RDS integration
- Backend development
